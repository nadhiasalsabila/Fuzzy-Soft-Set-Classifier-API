Fuzzy Soft Set Classifer

An algorithm that assigns a class label to an object, based on the object description. It is also said that the classifier predicts the class label. The object description comes in the form of a vector containing values of the features (attributes) deemed to be relevant for the classification task. Typically, the classifier learns to predict class labels using a training algorithm and a training data set. When a training data set is not available, a classifier can be designed from prior knowledge and expertise. Once trained, the classifier is ready for operation on unseen objects. Fuzzy Soft Set Classifier used mathematical model of vague qualitative or quantitative data, frequently generated by means of the natural language. The model is based on the generalization of the classical concepts of set and its characteristic function.
